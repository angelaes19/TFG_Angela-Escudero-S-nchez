# Trabajo de Fin de Grado - Programación y Simulación

Este repositorio contiene el desarrollo técnico y la programación correspondientes al Trabajo de Fin de Grado de Ángela Escudero Sánchez. El proyecto se centra en la implementación de sistemas de control mediante microcontroladores Arduino y el desarrollo de herramientas de simulación visual.

## Estructura del Repositorio

El contenido está organizado de forma iterativa, reflejando las distintas fases de desarrollo y pruebas del sistema:

### Implementaciones de Control (Arduino)
Se incluyen los códigos fuente (.ino) para las distintas versiones del hardware:
- **i1_P1 / i1_P2**: Versiones iniciales del sistema de control y lógica de procesos.
- **i2_P1 / i2_P2**: Iteraciones mejoradas con optimización de código y nuevas funcionalidades.
- **i3/proyecto1**: Versión consolidada del firmware para el prototipo actual.

### Herramientas de Simulación
Dentro de la carpeta `i3/simulador_punto` se encuentra un simulador desarrollado con tecnologías web para la validación visual de los patrones:
- **Lógica (JS)**: Control dinámico de la interfaz y comportamiento del sistema.
- **Interfaz (HTML/CSS)**: Estructura visual y estilos alineados con la identidad del proyecto.
- **Recursos**: Activos gráficos necesarios para la representación del simulador.

## Especificaciones Técnicas
- **Lenguaje**: C++ (Arduino IDE) para la lógica del microcontrolador.
- **Simulación**: HTML5, CSS3 y JavaScript vanilla para las herramientas de apoyo.
- **Hardware**: Diseñado para su ejecución en placas compatibles con el ecosistema Arduino.

## Objetivo del Proyecto
El objetivo principal de este código es gestionar la comunicación y el control de un sistema físico, permitiendo una interacción fluida entre los componentes electrónicos y proporcionando una interfaz de validación visual mediante el simulador adjunto.
